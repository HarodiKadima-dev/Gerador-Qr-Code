# 📱 Gerador de Código QR

Ferramenta simples, funcional e responsiva para criar códigos QR com qualquer texto, mensagem, ligação ou informação.

## ✨ Funcionalidades
✅ Entrada de texto livre  
✅ Geração rápida usando API pública  
✅ Funciona por **clique ou tecla Enter**  
✅ Bloqueia botão enquanto carrega — evita repetições  
✅ Botão de **download automático em PNG**  
✅ Tratamento de campos vazios e erros  
✅ Design moderno e adaptado para telemóvel

## 🛠️ Tecnologias usadas
- **HTML5** — estrutura
- **CSS3** — estilo, responsividade e efeitos visuais
- **JavaScript** — toda a lógica, eventos e integração com API
- API: `api.qrserver.com` — gratuita e sem necessidade de cadastro

## 📚 O que aprendi neste projeto
- Trabalhar com requisições assíncronas (`fetch`, `async / await`)
- Criar elementos e eventos **dinamicamente**
- Controlar estado dos botões (`habilitar / desabilitar`)
- Garantir segurança com `encodeURIComponent()`
- Melhorar experiência do utilizador com feedback visual
